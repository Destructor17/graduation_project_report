}{

\TestCase{10}{Проверка работы приложения rugpt2}{
    Среда выполнения кода загружена в формате AppImage с веб-сайта на персональный компьютер под управлением Kubuntu 23.10.
    Приложение rugpt2 скомпилировано на этом же компьютере. 
}{
    Проверяется процесс работы приложения rugpt2.
}{
    Приложение rugpt2 помещается в папку mods, которая располагается там же, где и среда выполнения кода.
    Запуск приложения происходит с помощью файлового менеджера.
}{
    При запуске среды выполнения кода появляется графический интерфейс приложения rugpt2, который предлагает пользователю ввести текст.
    После ввода этот текст будет продолжен с помощью генеративной нейронной сети \GPRSeeImageRef{test_rugpt2}.

    \GPRImage{test_rugpt2}{Графический интерфейс приложения rugpt2}{main/sections/3_implementation/images/rugpt2.png}{12cm}
}
